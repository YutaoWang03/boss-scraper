# `src/utils/` 工具层

## 职责

提供与业务弱耦合的通用能力，对应架构文档 **工具层**：薪资相关处理、人类行为模拟用的随机延迟等。PRD 提到薪资解密可能不完全准确，使用结果时需自行校验。

## 模块

| 文件 | 职责 |
|------|------|
| `salary.py` | Boss 薪资展示解密或规范化（字体映射等），供解析层使用 |
| `delay.py` | 随机等待区间，配合爬虫降低风控概率（与 PRD 6.3 一致） |

## 依赖说明

本层模块可被 `core` 或 `api` 引用；避免在此放置与页面结构强绑定的解析逻辑（应放在 `core/parser.py`）。
