# 产品需求文档 (PRD)

## 1. 产品概述

### 1.1 产品名称
Boss 直聘爬虫工具

### 1.2 产品定位
一款面向求职者的职位信息采集和分析工具，帮助用户快速获取目标岗位的市场行情。

### 1.3 目标用户
- 求职者：了解目标岗位的薪资范围和技能要求
- 招聘方：了解市场行情和竞争对手薪资
- HR：从业者了解市场趋势

## 2. 功能需求

### 2.1 核心功能

| 功能 | 优先级 | 描述 |
|------|--------|------|
| 职位搜索 | P0 | 根据关键词搜索职位 |
| 多城市支持 | P0 | 支持主要城市 |
| 数据导出 | P0 | 支持多种格式 |
| 风控检测 | P0 | 检测登录页、验证码 |
| 数据缓存 | P1 | 缓存已抓取数据 |
| 交互模式 | P1 | 支持交互式输入 |

### 2.2 数据分析功能（已提供基线实现）

| 功能 | 优先级 | 描述 | 实现位置 |
|------|--------|------|----------|
| 薪资分布 | P2 | 薪资区间分布图 | `analytics/chart.py` + `salary.py` |
| 技能分析 | P2 | 从描述/标签中提取技能（关键词表） | `analytics/skills.py` |
| 经验薪资曲线 | P2 | 薪资与经验关系 | `analytics/chart.py` + `salary.py` |
| 学历溢价分析 | P3 | 学历对薪资影响 | `analytics/chart.py` + `salary.py` |
| 公司规模分析 | P3 | 公司规模与薪资关系（描述启发式） | `analytics/company.py` + `chart.py` |

**使用方式：**

- 命令行：`python main.py --analyze <路径.csv|路径.json>`，可选 `--report-dir <根目录>`（默认在 `output/reports/<basename>/` 生成 `report.md` 与 PNG）。
- 交互：向导中选「数据分析（离线）」或「抓取并分析」（完成后对当次结果直接分析）。

## 3. 用户故事

### 3.1 场景一：求职者了解薪资行情
作为求职者，我想搜索"AI产品经理"岗位，了解杭州地区的薪资范围，以便评估自己的薪资期望是否合理。

**流程：**
1. 用户运行程序
2. 输入关键词"AI产品经理"
3. 选择城市"杭州"
4. 设置爬取页数
5. 获取职位数据

### 3. 2场景二：使用缓存数据
作为老用户，我想直接使用之前抓取的数据进行分析，不想重复爬取。

**流程：**
1. 用户运行程序
2. 系统检测到缓存数据
3. 询问是否使用缓存
4. 用户选择使用缓存
5. 加载已有数据

### 3.3 场景三：遇到风控停止
作为用户，我想在遇到验证码时自动停止，避免账号被封。

**流程：**
1. 程序检测到验证码
2. 自动保存已有数据
3. 提示用户风控信息
4. 退出程序

## 4. 数据需求

### 4.1 原始字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 岗位名称 | 文本 | 职位名称 |
| 薪资范围 | 文本 | 薪资文本（含加密）|
| 工作地点 | 文本 | 城市·区域 |
| 公司名称 | 文本 | 公司名 |
| 经验学历 | 文本 | 经验/学历要求 |
| 职位链接 | URL | 详情页链接 |
| 职位描述 | 长文本 | 完整描述 |

### 4.2 处理后字段

| 字段 | 类型 | 说明 |
|------|------|------|
| Salaryavg | 数值 | 平均薪资 |
| Years | 数值 | 工作年限 |
| Edulevel | 数值 | 学历等级 |
| Sizenum | 数值 | 公司规模数值 |
| Tags | 列表 | 技能标签 |

## 5. 分析功能详情

### 5.1 薪资区间分布图
- **输入**: 薪资范围字段
- **输出**: 直方图/饼图
- **解决的问题**: 了解市场行情均价
- **实现**: `salary_histogram.png`，分箱统计见 `report.md`

### 5.2 技能价值排行榜
- **输入**: 职位描述
- **输出**: 技能词频排行
- **解决的问题**: 简历该往哪个技术方向优化？
- **实现**: `skills_bar.png`；词表见 `analytics/skills.py` 中 `SKILL_KEYWORDS`

### 5.3 薪资-经验曲线
- **输入**: 薪资 + 工作年限
- **输出**: 折线图
- **解决的问题**: 我的经验值多少钱？
- **实现**: `salary_by_experience.png`（按「经验学历」字段解析经验桶）

### 5.4 学历溢价分析
- **输入**: 薪资 + 学历
- **输出**: 柱状图
- **解决的问题**: 考研/提升学历是否有经济回报？
- **实现**: `salary_by_education.png`

### 5.5 规模-薪资散点图
- **输入**: 薪资 + 公司规模
- **输出**: 散点图
- **解决的问题**: 大公司一定比小公司给得多吗？
- **实现**: `size_salary_scatter.png`（规模自职位描述正则推断，覆盖率见报告）

## 6. 非功能需求

### 6.1 性能
- 单页抓取时间：< 30秒
- 支持连续抓取：100+ 条/天

### 6.2 稳定性
- 异常捕获率：> 95%
- 缓存恢复：支持断点续传

### 6.3 安全性
- 随机延迟防封禁
- 验证码检测
- 登录态检测
